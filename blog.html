Question-1: What are the different ways to select an element in the DOM?

Ans:

  • Use getElementById()
  • Use getElementsByClassName()
  • Use querySelector()
  • Use querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:

  • innerHTML: Use it when working with HTML content
  • innerText: Use it when working with visible text
  • textContent: Use it when working with all text, including hidden elements

Question-3: What is event delegation in the DOM?

Ans:

Event delegation allows you to attach a single event listener to a parent element, which handles events from its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Ans:

Event bubbling is a behaviour where an event starts from the target (childs) element and moves upward through its parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans:

  • Create a new element: Use document.createElement("tag")
  • Add an element as the last child: Use parent.appendchild(child)
  • Add an element as the first child: Use parent.prepend(child)
  • Remove a specific child element: Use parent.removeChild(child)
  • Remove an element directly: Use element.remove()
